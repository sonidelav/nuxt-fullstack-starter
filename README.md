# Webpage Fullstack Starter Template
powered by Nuxt + NestJS

## Getting Started
Just clone me!
```shell
git clone https://github.com/sonidelav/nuxt-fullstack-starter.git
```

## Development

### Install
```shell
yarn install
```

### Development
```shell
yarn dev
```

### Build
```shell
yarn build
```

### Production
```shell
yarn start
```
